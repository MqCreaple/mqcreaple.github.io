盘点亿些Julia人性化的设计

Aug 11, 2021

Symbols

1. Greek Letters

Python:

alpha = 1
beta = 2
gamma = 3

Julia:

α = 1      # type "\alpha" and press [tab]
β = 2
γ = 3

2. constants

Python:

import math
print(math.pi)
print(math.e)

Julia:

println(π)     # "\pi"
println(ℯ)     # "\euler"

3. Operators

Python:

3*x**2+2*x+1
sqrt(x)

Julia:

3x^2+2x+1
√x  <==>  sqrt(x)

Arrays | Vectors

1. loop

Python:

import numpy as np
arr = np.array([1, 2, 3, 4, 5])
for a in arr:
    print(a)

Julia:

arr = [1, 2, 3, 4, 5]
for a ∈ arr       # a "\in" arr
    println(a)
end

2. vector operation

Python:

import numpy as np
np.add([1, 2, 3], 1)
np.add([1, 2, 3], [4, 5, 6])
np.multiply([1, 2, 3], 2)
np.multiply([1, 2], [3, 4])
def f(x):
    return 2 * x + 1
map(f, [1, 2, 3, 4])

Julia:

[1, 2, 3] .+ 1
[1, 2, 3] + [4, 5, 6]
[1, 2, 3] * 2
[1, 2] .* [3, 4]
f(x) = 2x + 1
f.([1, 2, 3, 4])

3. Matrix

Python:

import numpy as np
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
C = A.dot(B)
D = A.transpose()

Julia:

A = [1 2 ; 3 4]
B = [5 6 ; 7 8]
C = A * B
D = A'
